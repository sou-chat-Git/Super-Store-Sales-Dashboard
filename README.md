# Super-Store-Sales-Dashboard

## Data Preparation:
- Cleaned and transformed raw sales data
- Converted Order Date into month-wise and year-wise format
- Prepared data model for analysis



## Key KPIs:
- Total Sales
- Total Orders
- Total Profit
- Average Shipping Days

## Dashboard Visuals:
- Region-wise Sales
- Sales by Payment Mode
- Sales vs Consumer Type
- Sales by Product Category & Sub-Category
- Monthly Sales & Profit by Year
- Sales by Shipping Mode
- Top Performing States
- 15-Day Sales Forecast

This dashboard provides "Regions" as a slicer thus making it interactive for the user who can see the region wise sales for each chart. The dashboard solves questions like: 

1. What is the total profit?
2. What is the order count?
3. What is the total sales?
4. What is the average time taken for delivery?
5. Which payment mode incurs the highest sales?
6. Which region incurs the highest sales?
7. Which type of consumer makes the highest sales?
8. Which category and sub-category of products make the highest sales?
9. What type of shipping do customers generally prefer?
10. Monthly distribution of the sales done in 2019 and 2020
11. Monthly distribution of the profit gained in 2019 and 2020
12. Shows Statewise distribution ( more the sales more the bubble size )
13. Shows the sales forecast for 15 days
